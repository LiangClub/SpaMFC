# SpaMFC: Spatial Multi-Feature Clustering

**空间转录组亚群精细分群分析工具**

SpaMFC is a Python-based toolkit for fine-grained subcluster identification in spatial transcriptomics data.

## Key Features
- Integrate **spatial domain features**
- Integrate **gene expression features**
- Integrate **CNV features** (for malignant cells)
- Accurate and robust subcluster annotation

## Workflow
1. Read spatial transcriptomics data
2. Extract multi-modal features
3. Unsupervised multi-feature clustering
4. Visualization of spatial subclusters
